# sbcas_minicurso_fl_cnn
Implementação prática de Aprendizado Federado para triagem do câncer do colo do útero utilizando Flower, PyTorch, Docker e ResNet-18. O projeto explora FedAvg, privacidade de dados, cenários não-IID, avaliação global, explicabilidade (XAI) e análise de embeddings em imagens citopatológicas.
